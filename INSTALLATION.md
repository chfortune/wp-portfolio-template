# Installation Guide for WP Portfolio Template

## Prerequisites
To install this template, ensure you meet the following requirements:
- WordPress 5.0 or later
- PHP 7.4 or later

## Installation Methods
You can install the WP Portfolio Template using one of three methods:
### 1. FTP (File Transfer Protocol)
1. **Download** the template ZIP file from GitHub.
2. **Extract** the ZIP file on your local machine.
3. **Connect** to your web server using an FTP client (e.g., FileZilla).
4. **Upload** the extracted folder to `/wp-content/themes/` on your server.
5. **Log** into your WordPress dashboard and navigate to **Appearance > Themes**. Activate the WP Portfolio Template.

### 2. File Manager
1. **Download** the template ZIP file from GitHub.
2. **Extract** the ZIP file.
3. **Open** your hosting control panel (e.g., cPanel or Plesk).
4. **Navigate** to File Manager.
5. **Upload** the extracted folder to `/wp-content/themes/`.
6. **Log** into your WordPress dashboard and activate the WP Portfolio Template under **Appearance > Themes**.
